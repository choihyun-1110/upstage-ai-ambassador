# 🎯 AI 기반 맞춤 커리어 액션 추천 시스템
> Upstage AI Ambassador 2기 | 미션1: 노코드 기반 워크플로우 SNS 공유 (마감: 2025년 3월 31일)

---

## 📌 프로젝트 개요

사용자가 자신의 이력서, 자기소개서, 포트폴리오 등 **커리어 관련 PDF**를 업로드하면, AI가 "당신은 지금까지 이런 걸 해왔네요"라고 분석한 뒤, **매일 아침 9시에 맞춤형 액션(대외활동 추천, 채용 정보, 업계 뉴스, 오늘의 미션)을 이메일로 자동 발송**하는 n8n 기반 노코드 자동화 시스템.

### 핵심 가치
- **나를 아는 AI**: 단순 뉴스 요약이 아닌, 내 커리어 맥락에서 "왜 지금 이게 중요한지" 설명
- **액션 중심**: 읽고 끝나는 게 아니라, 오늘 실제로 할 수 있는 행동 제안
- **Upstage API 활용**: Document Parse로 PDF 분석, Solar LLM으로 개인화, Embeddings로 벡터 저장
- **노코드**: n8n으로 코딩 없이 전체 파이프라인 구성

## 🏗️ 시스템 아키텍처

```
[워크플로우 1: 프로필 등록 - 1회성]

사용자: 이름 + 이메일 + PDF 업로드 (이력서/자소서/포트폴리오)
    ↓
Upstage Document Parse
    → 텍스트 추출
    ↓
Upstage Solar LLM (커리어 분석)
    → "당신은 지금까지 이런 걸 해왔네요..."
    → 전공, 희망진로, 키워드, 강점, 성장포인트 추출
    ↓
┌──────────────────────────────────┐
│ (3가지 동시 실행)                   │
├──────────────────────────────────┤
│ 1. 로컬 JSON 저장                  │
│ 2. Upstage Embeddings → Supabase  │  ← pgvector (4096차원)
│ 3. 환영 이메일 발송 (분석 결과 포함)  │
└──────────────────────────────────┘


[워크플로우 2: 매일 오전 9시 자동 발송 - 평일 반복]

Schedule Trigger (09:00, 평일)
    ↓
저장된 프로필 읽기
    ↓
맞춤 데이터 수집 (병렬 fetch)
    → 🏆 대외활동/공모전 (Google News RSS)
    → 💼 인턴십/채용 (Google News RSS)
    → 📰 업계 뉴스 한/영 (Google News RSS)
    → 🎬 관련 유튜브 (Google News RSS)
    → 🔥 HackerNews Top10
    ↓
Upstage Solar LLM
    → 수집된 실제 데이터만 사용
    → 4개 섹션 + 오늘의 커리어 미션 생성
    ↓
Gmail 발송 (HTML 이메일)
```

## 📬 이메일 구성

### 등록 직후 환영 이메일
- AI가 본 당신 (career_analysis)
- 강점 목록
- 성장 포인트

### 매일 아침 액션 리포트
```
📬 [홍길동]님의 오늘 커리어 액션 — 2026년 3월 26일 목요일

🏆 오늘 신청할 대외활동/공모전
  1. [공모전명] + 왜 지원해볼만한지

💼 놓치면 안될 채용/인턴 정보
  1. [채용공고] + 왜 지원해볼만한지

📰 오늘 꼭 읽어야 할 업계 뉴스
  1. [뉴스 제목] + 핵심 요약 + 나와의 관련성

🎯 오늘의 커리어 미션
  → 구체적이고 실천 가능한 행동 1가지
```

---

## 🛠️ 기술 스택

| 구분 | 도구 |
|------|------|
| **자동화** | n8n (로컬, Node v20 LTS via nvm) |
| **문서 분석** | Upstage Document Parse API |
| **AI 분석/요약** | Upstage Solar LLM (solar-pro) |
| **벡터 임베딩** | Upstage Embeddings (solar-embedding-1-large, 4096차원) |
| **벡터 DB** | Supabase pgvector |
| **데이터 수집** | Google News RSS, HackerNews RSS |
| **이메일 발송** | Gmail SMTP (n8n emailSend 노드) |

---

## 🔑 설정 필요한 n8n 크레덴셜
